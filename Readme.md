#  Login Page

A Simple, clean, and responsive Login & Sign Up web interface built using **HTML, CSS, and JavaScript**. 

## ✨ Features
- Modern centered card design
- Username, Email & Password fields
- Show/Hide password toggle
- Smooth button styling
- Responsive layout

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
